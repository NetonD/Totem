## Requisitos

- Pillow >= 5.1.0
- Pyodbc >= 4.0.23
- Tkinter >= 8.6.7

## Instalação
1. Baixe os arquivos ou clone o repositorio
2. Execute o data_base_tables.sql em seu Sql server
3. Insira o login, dns do servidor e senha nas variaveis LOGIN, SENHA, SERVIDOR
